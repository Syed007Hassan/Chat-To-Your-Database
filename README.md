# Chat to your database

This is an experimental app to test the abilities of LLMs to query SQL databases using natural language.
To use it, you should add your OPENAI_API_KEY to the .env.local file.

## Installing the app

```bash!
npm install
```

## Running the app

```bash
npm run dev
```
